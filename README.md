# AI 驱动的可视化产品设计 - 演示项目

这是一个关于「用 AI 画原型、写文档」的分享演示项目。

## 快速开始

### 方式一：在线访问（推荐）

直接访问部署好的链接即可。

https://songwb89.github.io/ai-ppt-web/presentation/index.html

### 方式二：本地运行

由于浏览器安全限制，直接双击打开 HTML 文件可能会导致部分功能异常（如按键翻页失效、文档无法渲染）。

推荐使用以下方式启动本地服务：

**方法 1：使用 Node.js**

```bash
# 在项目根目录执行
npx serve
```

然后访问显示的地址

**方法 3：使用 VS Code / Kiro**

1. 安装 Live Server 插件
2. 右键点击 `index.html`
3. 选择「Open with Live Server」

## 演示操作说明

### 翻页控制

- `←` / `→` 键：上一页 / 下一页
- 也可以点击底部的翻页按钮

### 界面控制

- `T` 键：打开/关闭目录
- `F` 键：进入/退出全屏
- 鼠标移到屏幕底部：显示控制栏

### 素材查看

- 部分页面右上角会显示素材图标（视频、页面、链接）
- 点击图标可预览对应素材
- `ESC` 键：关闭预览弹窗
