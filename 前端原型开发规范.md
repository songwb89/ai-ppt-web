# 前端原型开发规范

在开始画原型之前，请告知 AI 以下技术选型，确保输出一致性。

---

## 1. CSS 框架

| 选项 | CDN 引入 | 说明 |
|------|----------|------|
| **Tailwind CSS**（推荐） | `<script src="https://cdn.tailwindcss.com"></script>` | 原子类，快速开发 |
| 原生 CSS | 无需引入 | 简单项目 |
| Bootstrap | `<link href="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/css/bootstrap.min.css">` | 需要现成组件 |

**默认选择：Tailwind CSS**

---

## 2. 图标库

| 选项 | CDN 引入 | 使用方式 |
|------|----------|----------|
| **Heroicons**（推荐） | 内置于 Tailwind，直接用 SVG | `<svg>...</svg>` |
| Font Awesome | `<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.0/css/all.min.css">` | `<i class="fa-solid fa-home"></i>` |
| Material Icons | `<link href="https://fonts.googleapis.com/icon?family=Material+Icons">` | `<span class="material-icons">home</span>` |
| Lucide Icons | `<script src="https://unpkg.com/lucide@latest"></script>` | `<i data-lucide="home"></i>` |

**默认选择：Font Awesome**（图标丰富，使用简单）

---

## 3. 字体

| 选项 | CDN 引入 |
|------|----------|
| **系统默认字体** | 无需引入，使用 `font-family: system-ui, sans-serif` |
| 思源黑体 | `<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC&display=swap">` |
| Inter | `<link href="https://fonts.googleapis.com/css2?family=Inter&display=swap">` |

**默认选择：系统默认字体**（加载快，中文兼容好）

---

## 4. UI 组件库（可选）

| 选项 | 适用场景 | CDN 引入 |
|------|----------|----------|
| **无组件库** | 简单原型，自己写 | - |
| Element Plus | Vue 项目 | 需 Vue 环境 |
| Ant Design | React 项目 | 需 React 环境 |
| DaisyUI | Tailwind 增强 | `<link href="https://cdn.jsdelivr.net/npm/daisyui@4/dist/full.min.css">` |

**默认选择：无组件库**（纯原型不需要）

---

## 5. 图表库（如需要）

| 选项 | CDN 引入 |
|------|----------|
| **ECharts**（推荐） | `<script src="https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js"></script>` |
| Chart.js | `<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>` |

---

## 6. 动画库（可选）

| 选项 | CDN 引入 |
|------|----------|
| Animate.css | `<link href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css">` |
| AOS | `<link href="https://unpkg.com/aos@2/dist/aos.css">` + `<script src="https://unpkg.com/aos@2/dist/aos.js"></script>` |

---

## 7. 其他约定

| 项目 | 约定 |
|------|------|
| **响应式** | 默认支持，使用 Tailwind 的 `sm:` `md:` `lg:` 前缀 |
| **暗色/亮色** | 默认亮色，如需暗色请说明 |
| **假数据** | AI 自动生成，中文内容 |
| **文件结构** | 单页原型用单个 HTML，多页用文件夹组织 |
| **注释** | 关键区块添加中文注释 |

---

## 快速模板

告诉 AI 时可以直接说：

> "用 Tailwind CSS + Font Awesome，系统默认字体，需要 ECharts 图表，暗色主题"

或使用默认配置：

> "用默认规范"

默认规范 = Tailwind CSS + Font Awesome + 系统字体 + 亮色主题

---

## HTML 模板头部

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题</title>
  <!-- Tailwind CSS -->
  <script src="https://cdn.tailwindcss.com"></script>
  <!-- Font Awesome -->
  <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.0/css/all.min.css" rel="stylesheet">
</head>
<body>
  <!-- 内容 -->
</body>
</html>
```
