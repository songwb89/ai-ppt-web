# 灵感画廊页面 - AI 生成提示词

## 页面概述

做一个 AI 图片分享社区的「灵感画廊」页面，采用现代图片社区网站的设计风格，极简、大留白、让图片成为主角。

## 页面结构

### 顶部导航栏

- 左侧：Logo，使用「Muse」这个英文单词，用艺术感的字体
- 中间：三个导航标签「灵感画廊 / 我的作品 / 开始创作」，灵感画廊为当前选中状态
- 右侧：用户头像（圆形）+ 用户名

### 内容区域

- 瀑布流布局展示图片
- 图片宽度一致，高度不等（保持原图比例）
- 响应式：桌面 4 列，平板 3 列，手机 2 列

### 图片卡片

- 图片使用 Unsplash 随机图片
- 鼠标悬停时显示：
  - 左下角：作者头像（小圆形）+ 用户名
  - 右下角：空心爱心图标 + 点赞数
  - 图片上方加一层渐变遮罩，让文字更清晰
- 点击爱心可点赞，爱心变成红色实心，数字 +1

### 滚动加载

- 初始加载 20 张图片
- 滚动到底部自动加载更多，每次加载 12 张
- 总共 44 张图片，加载完毕后显示「已经到底啦~」

## 风格要求

- 整体风格：极简、干净、大留白、让图片成为主角
- 配色：黑白灰为主，彩色只来自图片本身
- 导航栏：白色背景，细字体，无边框
- 卡片：无边框，悬停时轻微放大 + 阴影加深

## 交互动效

- 页面加载时图片依次淡入
- 图片悬停：阴影变深 + 信息浮现
- 点赞：爱心有心跳动画
- 滚动加载时显示 loading 动画

## 技术方案

- 单 HTML 文件实现，
- 用 Tailwind CSS + Font Awesome
