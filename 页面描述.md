创建一个教师管理界面，默认规范 = Tailwind CSS + Font Awesome + 系统字体 + 亮色主题。现代化后台管理系统风格。
顶部显示面包屑导航：系统首页/教师管理
1. 教师列表：教师列表字段含 序号、姓名、手机号、邮箱、任职院校、关联院校、创建时间、最后登录时间、启用状态（开关，支持操作）、操作（含编辑、重置密码、修改院校）。姓名、创建时间、最后登录时间支持排序。非必填项，无内容时，显示“--”占位符。筛选条件支持姓名、手机/邮箱、关联院校、任职院校、创建时间、启用状态。列表底部有分页组件。
2. 添加教师：字段含邮箱（必填）、密码（必填）、姓名（必填）、关联院校（必填、下拉选择）、任职院校（下拉选择）、任职专业、职位、简介（文本域）。
3. 编辑教师：无密码字段；邮箱、关联院校 字段禁用。
4. 重置密码：点击填写新的密码进行重置。
5. 修改院校：点击可以变更关联院校
